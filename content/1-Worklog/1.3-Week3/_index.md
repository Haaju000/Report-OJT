### Week 3 Objectives:

* Master how to manage data on Amazon S3 including buckets, access rights, versioning, lifecycle and security
* Know how to deploy static websites on S3 and practice building a complete storage system with permissions, cost optimization and data lifecycle management.

### Week 3 Achievements:

* Understand and implement S3 features: bucket, object, permission, versioning, storage classes and lifecycle.

* Deploy static website on S3 and configure public/private bucket according to security standards.

* Complete the practical lab including 1 private bucket, 1 public hosting bucket and 1 lifecycle rule to automatically transfer data to Glacier after 30 days.
