###  [Blog 1 - University of California Irvine backs up petabytes of research data to AWS](3.1-Blog1/)
The University of California, Irvine (UCI) used rclone and multiple Amazon Web Services services to build a centralized backup system, protecting more than 5 PB of research data (over 250 million files) from hundreds of lab servers.

###  [Blog 2 - The frugal HPC architect – ensuring effective FinOps for HPC workloads at scale](3.2-Blog2/)
When running HPC workloads on Amazon Web Services (AWS), you should consider cost as an essential requirement and use tools + strategies (Spot, Savings Plan, choosing the right instance, monitoring usage...) to optimize performance - cost.

###  [Blog 3 - Enhance API throttling visibility with Amazon QuickSight](3.3-Blog3/)
Amazon QuickSight lets you create a centralized dashboard to view and analyze all API “throttle” events from multiple AWS accounts and regions — making it easy to detect throttled APIs and optimize your API call logic.
