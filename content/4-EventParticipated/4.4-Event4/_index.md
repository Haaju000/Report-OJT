# Summary Report: “AWS Cloud Mastery Series #2: ​DevOps on AWS”

### Event Objectives

- DevOps and Cloud tutorials on AWS

- Help understand DevOps mindset, tools and professional processes when developing software on AWS

- Provide practical knowledge to better understand

### Speakers

**AWS ​DevOps Specialists**

### Key Highlights

- Instructions on using Amazon Web Services (AWS) such as CodeCommit, CodeBuild, CodeDeploy, CodePipeline

- Introduction and demo of using IaC with AWS CloudFormation and AWS CDK

- Basic Docker tutorial, using registry (ECR) and deployment to AWS container services

- Setting up monitoring, logging, tracing: using Amazon CloudWatch, AWS X-Ray and learning how to build a full observability system

- Career path consulting

### Key Takeaways

- Học được cách sử dụng IaC thông qua AWS CloudFormation và AWS CDK để định nghĩa và triển khai hạ tầng cloud bằng mã thay vì thủ công

- Biết cách dùng các công cụ của AWS như AWS CodeCommit, AWS CodeBuild, AWS CodeDeploy và AWS CodePipeline

- Học cách thiết lập monitoring + logging + tracing cho ứng dụng và hệ thống


### Event Experience

Attending the **“AWS Cloud Mastery Series #2: ​DevOps on AWS”** workshop was extremely valuable, giving me a comprehensive view of modernizing applications and databases using advanced methods and tools. Key experiences included:

- Tham gia demo và thực hành trực tiếp

- Khám phá container và cloud-native services

- Trải nghiệm thiết lập giám sát và observability
