# Bài thu hoạch "DATA SCIENCE ON AWS"

### Mục Đích Của Sự Kiện

- Giúp người hiểu rõ tầm quan trọng của Cloud + Data Science

- Mở rộng cơ hội nghề nghiệp hoặc định hướng học tập cho những ai muốn theo đuổi lĩnh vực Data Science, Cloud, Machine Learning, Big Data

- Trải nghiệm thực tế pipeline Data Science

- Biết cách xử lý và làm sạch dữ liệu từ dataset IMDb với AWS Glue

### Danh Sách Diễn Giả

- **Văn Hoàng Kha** - Cloud Solutions Architect, AWS Community Builder
- **Bạch Doãn Vương** - Cloud Develops Engineer, AWS Community Builder

### Nội Dung Nổi Bật

- Giới thiệu tổng quan về pipeline Data Science trên AWS

- Thực hành xử lý và làm sạch dữ liệu từ dataset IMDb với AWS Glue

- Huấn luyện và triển khai mô hình Sentiment Analysis với SageMaker

- So sánh về chi phí, hiệu năng của Cloud và On-premise

- Hướng dẫn và thực hành để có thể tự làm project nhỏ sau workshop

### Những Gì Học Được

- Hiểu được khái niệm và quy trình cơ bản của Data Science

- Phân biệt được sự khác nhau giữa làm data và Machine Learning trên cloud và on-premise

- Hiểu được cách xây dựng Data Pipeline và Machine Learning trên cloud


### Trải nghiệm trong event

Tham gia workshop **“DATA SCIENCE ON AWS”** là một trải nghiệm rất bổ ích, giúp tôi có cái nhìn toàn diện về cách hiện đại hóa ứng dụng và cơ sở dữ liệu bằng các phương pháp và công cụ hiện đại. Một số trải nghiệm nổi bật:

- Xây dựng mô hình Machine Learning

- Sử dụng AWS S3 để lưu trữ dữ liệu

- Sử dụng AWS SageMaker để huấn luyện và deploy Machine Learning model
