# Summary Report: “DATA SCIENCE ON AWS”

### Event Objectives

- Help people understand the importance of Cloud + Data Science

- Expand career opportunities or study orientation for those who want to pursue the fields of Data Science, Cloud, Machine Learning, Big Data

- Experience the real Data Science pipeline

- Know how to process and clean data from the IMDb dataset with AWS Glue

### Speakers

- **Van Hoang Kha** - Cloud Solutions Architect, AWS Community Builder
- **Bach Doan Vuong** - Cloud Developments Engineer, AWS Community Builder

### Key Highlights

- Overview of Data Science pipeline on AWS

- Practice processing and cleaning data from IMDb dataset with AWS Glue

- Train and deploy Sentiment Analysis model with SageMaker

- Compare cost and performance of Cloud and On-premise

- Instructions and practice to be able to do small projects after the workshop 

### Key Takeaways

- Hiểu được khái niệm và quy trình cơ bản của Data Science

- Phân biệt được sự khác nhau giữa làm data và Machine Learning trên cloud và on-premise

- Hiểu được cách xây dựng Data Pipeline và Machine Learning trên cloud


### Event Experience

Attending the **“DATA SCIENCE ON AWS”** workshop was extremely valuable, giving me a comprehensive view of modernizing applications and databases using advanced methods and tools. Key experiences included:

- Build Machine Learning models

- Use AWS S3 to store data

- Use AWS SageMaker to train and deploy Machine Learning models
